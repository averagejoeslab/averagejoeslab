## Quick Reference

- **Stack**: Next.js 16, React 19, Tailwind CSS 4, TypeScript
- **Build**: `npm run build` (static export to `out/`)
- **Dev**: `npm run dev` (fetches blog content first)
- **Lint**: `npm run lint`
- **Type check**: `npm run type-check`
- **Blog content**: Fetched from `averagejoeslab/averagejoeslab-blog` at build time
- **Papers data**: Committed JSON in `data/`, synced from Notion daily
